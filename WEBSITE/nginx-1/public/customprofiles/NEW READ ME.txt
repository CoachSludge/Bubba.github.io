# 📄 README: Local File Update Instructions

## 🚀 Overview
This update ensures that all local files are properly configured for **bubba.industries**. Please follow the steps carefully to avoid issues.

## 🛠️ Recommended Editor
📌 **Use Notepad++** for efficient **mass replace** functionality.

---

## 🔄 Required Changes

### 📌 Username Replacement
Replace:
```
%5Busername%5D  
```
➡ **with**  
```
username  
```

---

### 🌐 Update Paths
🔴 **No Longer Required:**  
Remove:
```
/_next/  
```
➡ **Replace with:**  
```
https://guns.lol/_next/  
```

⚠ **Potential Cloudflare Issues**  
🔄 Replace:
```
https://assets.guns.lol/_pow.js  
```
➡ **with**  
```
https://bubba.industries/_next/static/main/_pow.js  
```

🔄 Replace:
```
https://sa.guns.lol/latest.js  
```
➡ **with**  
```
https://bubba.industries/_next/static/main/latest.js  
```
